# Capturas opcionais

Este diretório é reservado para imagens oficiais do produto.

O README principal não depende de screenshots enquanto elas não existirem. Quando forem adicionadas, use imagens reais do app, sem dados pessoais e sem paths locais do ambiente de desenvolvimento.

Sugestão de arquivos:

- `library.png`: biblioteca, filtros e edição em massa.
- `chat.png`: conversa com a Dona Flora usando o acervo como contexto.
- `settings.png`: configuração de pasta local e provider de IA.
- `trails.png`: trilhas de leitura e progresso.

Evite screenshots com caminhos locais, nomes de arquivos privados, tokens ou conteúdo pessoal do Obsidian.
